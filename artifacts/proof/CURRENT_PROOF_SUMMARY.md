## Important limitations

- Mock-only deterministic runtime — no real LLM integration.
- No real external tool execution (execute_bounded_tool is policy-gated mock).
- Not production-ready.
- SimWorld is a synthetic regression harness, not general intelligence proof.
- NL benchmark routes all current 18 scenarios correctly; does not prove generalisation.
- Memvid backend is stubbed (returns NotImplemented).
- Python is legacy/reference only.
- No autonomous execution, no sentience, no consciousness, no AGI.
- Operational Pressure Modulator: pressure bias applied in both standard and NL SimWorld through run_cycle_with_bias(). Replay tracks counters; full pressure-state vector reconstruction is not yet implemented.
- Proof artifacts must be regenerated by CI; committed snapshots are receipts.
